/contrevenants get

get

Retourne la liste des restaurants ayant reçu une contravention entre les dates spécifiées, et le nombre de contraventions reçu.

get /contrevenants

Retourne la liste des restaurants ayant reçu une contravention entre les dates spécifiées, et le nombre de contraventions reçu.

  • Request
  • Response

Query Parameters

  • du: required(string)

    Date de début (format:"YYYY-MM-DD").

  • au: required(string)

    Date de fin (format:"YYYY-MM-DD").

HTTP status code 200

Succès de la requête. La liste des contrevenants et leurs contraventions.

Body

Media type: application/json

Type: array of object

Items: Contrevenant

  • etablissement: required(string)
  • nb_contraventions: required(integer)
